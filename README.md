# CommandShift Skills

Claude Code skills for automating business workflows — SEO, ads, design, content, email, and more.

## Skills

| Skill | What It Does | Key Tools |
|-------|-------------|-----------|
| [seo-audit](./seo-audit/) | Full SEO audit with 100-point scoring and actionable fixes | Firecrawl, WebFetch |
| [ad-campaign-generator](./ad-campaign-generator/) | Multi-platform ad campaigns (Facebook, Google, Instagram) | Firecrawl |
| [viral-script-agent](./viral-script-agent/) | Research creators, extract viral hooks, generate ready-to-film scripts | Web Search, Firecrawl |
| [website-design-clone](./website-design-clone/) | Clone any website's design system into HTML/CSS/Tailwind | Firecrawl, Playwright |
| [blog-writer](./blog-writer/) | SEO-optimized blog posts with real research (anti-AI-slop) | Web Search, Firecrawl |
| [inbox-manager](./inbox-manager/) | Gmail triage, auto-categorize, draft responses, bulk cleanup | Gmail MCP |
| [ai-news-briefing](./ai-news-briefing/) | Personalized AI news briefing from the last 30 days | Web Search |
| [design-templates](./design-templates/) | Create social graphics, thumbnails, banners | Canva MCP, Image Gen |
| [image-enhancer](./image-enhancer/) | 7-layer prompt framework to upgrade AI images to professional quality | Image Generation |
| [remote-control-setup](./remote-control-setup/) | Control Claude Code from your phone via SSH + task queue | System Tools |

## Installation

Copy any skill folder to your Claude Code commands directory:

```bash
cp -r seo-audit/ ~/.claude/commands/seo-audit/
```

Or reference directly in your CLAUDE.md:

```
Skills: ~/content-engine/skills/seo-audit/skill.md
```
