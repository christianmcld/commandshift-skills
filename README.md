## Installation

Copy any skill folder to your Claude Code commands directory:

```bash
cp -r seo-audit/ ~/.claude/commands/seo-audit/
```

Or reference directly in your CLAUDE.md:

```
Skills: ~/content-engine/skills/seo-audit/skill.md
```
